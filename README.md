# 👜 Cartify – Full Stack E-Commerce Bag Store

Cartify is a modern full-stack e-commerce web application built for selling stylish bags online.  
It includes user authentication, admin product management, dynamic cart functionality, and a responsive UI built with Tailwind CSS.

---

## 🚀 Features

### 🛍 User Features
- Browse all bag products
- Dynamic pricing with discount logic
- Add products to cart
- View total cart amount
- Session-based authentication
- Clean and responsive UI

### 🔐 Admin Features
- Secure admin login
- Create new products
- Upload product images
- Set product price and discount
- Customize product UI colors dynamically

---

## 🛠 Tech Stack

### Frontend
- EJS (Embedded JavaScript Templates)
- Tailwind CSS
- Remix Icons

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Other
- Session-based authentication
- Base64 image handling
- RESTful routing
- MVC architecture
